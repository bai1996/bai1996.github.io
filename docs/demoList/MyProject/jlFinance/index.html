银行理财

我国商业银行提供的理财产品分为保本固定收益产品、保本浮动收益产品与非保本浮动收益产品三类。

证券公司理财

证券理财一般包括股票、基金、商品期货、股指期货、外汇期货等，个人或机构投资者可以按照其不同需求及投资偏好选择不同理财工具。

保险理财

保险理财更加倾向长期性，着重解决较长时间后的教育规划和养老规划，同时解决意外、医疗等保障问题。

银行起源

一般认为最早的银行是意大利1407年在威尼斯成立的银行。

其后，荷兰在阿姆斯特丹、德国在汉堡、英国在伦敦也相继设立了银行。十八世纪末至十九世纪初，银行得到了普遍发展。

银行分类

  • 中央银行：如，中国人民银行
  • 监管机构：如，中国银行业监督管理委员会
  • 自律组织：如，中国银行业协会
  • 银行业金融机构：包括政策性银行、大型商业银行5家
  • 全国性股份制中小型商业银行12家
